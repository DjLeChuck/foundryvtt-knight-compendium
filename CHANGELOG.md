# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

### Fixed

- fix the 250GP Warlock evolution
- fix missile pod and rocket pod permanent state

## [12.1.3] - 2024-10-26

### Fixed

- fix: Babele loading for Foundry v12+
- fix: typo in one overdrive

## [12.1.2] - 2024-10-23

### Fixed

- fix: add missing `translations/` directory into module zip creation

## [12.1.1] - 2024-10-23

### Fixed

- fix: small translation fixes

## [12.1.0] - 2024-10-23

Thank you @Zakarik for all the changes of this release!

### Added

- feat: add FM4 Codex
- feat: add Babele in order to support English
- feat: all compendiums English translations

### Fixed

- fix: several items adjustements

## [12.0.1] - 2024-06-19

### Fixed

- fix(armours-base): missing effect and wrong violence values on Wizard
- fix(armours-atlas): wrong espoir values on Berzerk

## [12.0.0] - 2024-05-29

### Added

- FoundryVTT v12 compatibility

## [11.1.3] - 2024-04-02

### Fixed

- fix: energy cost on Warlock

## [11.1.2] - 2024-04-09

Thank you @sdurrheimer for the following fixes

### Fixed

- fix(weapons-rare): change 'Pavois - Charge' to be of 'contact' type
- fix(modules-rare): enable 'Erzatz Rogue' effect on 'Camouflage Optique'
- fix(modules-rare): enable 'Erzatz Bard' effect on 'Déguisement'
- fix(weapons-rare): add boost effects on 'Paire de pistolets ouragan'

## [11.1.1] - 2024-03-25

### Fixed

- Missing `Espoir` bonus on Berserk armour evolutions

## [11.1.0] - 2024-02-23

### Added

- Add legend meta-armour and ultimate capacities

### Fixed

- Wrong defense and reaction on some creatures

## [11.0.1] - 2023-12-01

### Fixed

- Misspell prototype token name of "Enfants du masque"

## [11.0.0] - 2023-11-12

### Added

- (Dev.) GitHub action workflow for module release

### Changed

- (Dev.) Commit JSON files for packs and use FoundryVTT cli to pack/unpack them

### Removed

- Remove support for FoundryVTT v10

### Fixed

- Description of heroic capacity "Colère du Juste"

## [10.1.1] - 2023-09-07

### Fixed

- Bad trait on "Épée bâtarde"
- Wrong damages on Monk capacities

### Changed

- Global update for the system 3.21

## [10.1.0] - 2023-06-21

### Added

- RollTable for serious injuries

### Fixed

- Missing malus on the serious injury "Oreille éclatée"

## [10.0.4] - 2023-06-11

### Added

- New capacity on the "Sorcerer"

### Fixed

- Wrong energy cost on the "Phase" special of the "Sorcerer"

## [10.0.3] - 2023-06-10

### Added

- Compendium packs for v11
- Add 37 new actors
- Add some illustrations on actors

### Fixed

- Only change pack visibility if admin

### Changed

- Group all NPCs into one compendium

## [10.0.2] - 2023-04-17

### Fixed

- Wrong impulsion count on Warmaster last evolution
- Wrong description for the "Céa" mode on the Monk
- Missing special "Récolte du flux" on the Psion

### Changed

- Re-organize actors from base book into locations instead of "La Geste" arcs

## [10.0.1] - 2023-04-09

### Modified

- Update all overdrives to reflect new level system

## [10.0.0] - 2023-04-08

### Modified

- Follow Foundry major version
- Update all module to reflect new level system

### Fixed

- Swap "Rayon d’énergie" damage dice for level 2 and 3 on Monk’s armor

## [0.0.20] - 2023-03-04

### Fixed

### Fixed

- Add Ranger longbow data

## [0.0.17] - 2022-12-21

### Added

- Images for 4 Nodachi NPC

### Modified

- Reduce length of compendium name for `La Geste de la fin des temps`
- Change `Attaque sur casque niv. 2` as a damage bonus on contact attacks instead of a weapon

## [0.0.16] - 2022-11-20

### Modified

- Multiple adjustments on Berserk
- Wrong damages bonus on Monk last evolution
- Wrong energy bonus on Warmaster armour

## [0.0.15] - 2022-11-13

### Fixed

- Remove undesirable unicode characters

### Modified

### Fixed

- Duplicate IDs between some compendium items

## [0.0.11] - 2022-11-08

### Added

- Official NPCs images
- Missing module "Option grenades intelligentes avancées"
- Missing bonus on multiple modules
- Set token texture on actors

### Fixed

- Bad configuration on last level of Warrior
- Rework Ranger entry to get in line with the new treatment in the system

### Added

- Official bestiary images
- NPCs of "La Geste de la fin des temps"
- Vehicles
- Serious injuries

- Merge one hand/two hands weapons
- Merge multiple ammo weapons
- Missing NPC capacities
- Missing creatures and humans weapons

### Added

- Overdrives
- Sections disadvantages
- Missing "Vrai artiste" advantage for Knights

### Modified

- Multiple adjustments on compendium (see [#4](https://github.com/DjLeChuck/foundryvtt-knight-compendium/issues/4))

### Added

- Mecha-armours actors

- Advantages and disadvantages for Knight and AI
- Destiny cards with associated roll table
- Heroic capacities
- Creatures weapons

- The Knight NPCs
- La Black Cell NPCs
- Le Nodachi NPCs
- Les Neufs NPCs

### Updated

- Multiple manual enrichments of compendiums
